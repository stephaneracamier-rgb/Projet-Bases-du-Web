Partie 1 - Découvrez le web et ses origines

Web

Définition :

Comme une toile d'araignée, avec ses nombreux fils connectés, le web est un ensemble de pages, accessibles depuis un lien hypertexte comme ceci.

Le lien hypertexte est probablement la plus grande invention à la base du Web. Le fait d'être sur une page et de cliquer sur un mot, ou même une image, pour en voir une autre vous paraît peut-être basique aujourd'hui, mais c'est la base d'une vraie révolution !

Au fait, avez-vous déjà regardé de près un lien hypertexte ?

Voici celui de la page d'accueil de Google : www.google.fr. Le « www » que vous voyez présent dans de nombreuses adresses signifie World Wide Web, soit « toile mondiale ».

Grâce aux liens hypertexte, chaque page sur le web est liée à d'autres pages. Par exemple, la page d'accueil d'OpenClassrooms fait des liens vers des cours, vers la page de connexion aux comptes, etc. De même, quand vous faites une recherche sur Google, quel est le résultat ? Des liens vers plein de sites !

Un autre exemple : Wikipédia. Quand vous consultez une page Wikipédia, il est fréquent d'avoir des liens vers d'autres articles, qui eux-mêmes pointent vers d'autres articles et ainsi de suite.

Internet

Définition :

Internet, c'est le réseau qui permet de communiquer entre ordinateurs. Sur ce réseau, on trouve plusieurs services :

Le Web (le plus connu d'entre eux) : vous ouvrez un navigateur web pour y aller, comme Google Chrome, Firefox, Internet Explorer, Edge, Safari...

  • Les e-mails : pour échanger des messages. Il s'agit tout simplement de courrier électronique.
  • Les newsgroups : moins connus, ils sont l'ancêtre des forums et permettent de discuter à plusieurs en postant des messages.
  • Le FTP : un moyen d'échanger des fichiers entre ordinateurs.
  • Etc.

On confond souvent le Web avec le reste, car tous ces services convergent vers le Web. Par exemple auparavant, on utilisait obligatoirement un logiciel dédié pour les e-mails (Mozilla Thunderbird, Apple Mail, Outlook...). Si ces logiciels existent toujours, aujourd'hui, on passe par le Web pour accéder à ses e-mails ! On va sur le site de Gmail, par exemple.

Le Web sert donc de porte d'entrée à la plupart des services aujourd'hui.

Le Cloud

Définition :

On parle souvent de Cloud... qu'est-ce que cela signifie concrètement ? Ce n'est pas évident, car c'est un terme (très) utilisé en marketing. Essayons d'y voir plus clair !

Comme pour le mot web, « Cloud » est un mot en anglais qui signifie « nuage ». Ceci nous donne un indice : imaginez un nuage dans le ciel, capable de maintenir de l'eau et de la relâcher sous forme de pluie à n'importe quel endroit.

Le « cloud » informatique opère de façon similaire, mais il conserve plutôt vos données et services. Prenons l'exemple de vos photos. Lorsque vous les stockez dans le cloud, vous évitez de les garder en local sur votre ordinateur. Le cloud est accessible de partout avec une connexion internet et s'adapte à vos besoins : quand vous avez besoin de plus d'espace de stockage, il s'agrandit, évoquant ainsi la forme changeante d'un nuage.

Le cloud est composé des fameux services fournis via une interface web dont je vous parlais ci-dessus. Par exemple, on pourrait dire que Gmail est un service d'email sur le Cloud.

Vous utilisez chaque jour des services cloud :

  • Des services d'email : Gmail, Outlook, etc.
  • Des services de stockage de documents : Dropbox, Google Drive par exemple ;
  • Des services de messagerie : Slack, Discord, Messenger ;
  • Etc.

Le Web sert alors de passerelle à de nombreux services, sous la dénomination « cloud ».

W3C

Définition :

Création en 1994 du World Wide Web Consortium ou W3C, un organisme qui a pris le relais de Tim Berners-Lee pour faire évoluer les technologies du web (HTML, CSS, PNG, XML et autres noms barbares).

Partie 2 - Distinguez les langages du Web

Langages client

Définition

Les langages client servent à fournir des informations de base pour la construction des sites web. C'est un peu comme des plans d'architecte. Ces langages sont principalement :

  • HTML, qui permet de renseigner le contenu du site (textes ou images...) ;
  • CSS, qui permet de définir le visuel du site (les couleurs, la police, les tailles...);
  • JavaScript, qui permet d'ajouter de l'interaction sur le site (menus déroulants, affichage de messages lors de la validation de formulaires...).

Ces trois langages sont désormais indispensables à la réalisation de tous les sites web. On dit que ce sont des langages client ou encore des langages frontend, car ils sont lus par les machines des clients à travers les navigateurs web.

Nous n'apprendrons pas à nous en servir dans ce cours, mais vous pourrez enchaîner avec d'autres cours accessibles aux débutants si vous voulez vous entraîner à les utiliser !

Navigateurs

Définition

Pour accéder aux sites web, on a besoin de navigateurs web. Il existe notamment :

  • Google Chrome
  • Mozilla Firefox
  • Edge (le successeur d'Internet Explorer)
  • Safari

Le rôle des navigateurs est de traduire les langages HTML, CSS et JavaScript sous la forme de sites web utilisables par tout le monde.
